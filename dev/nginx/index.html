Testing

To run test probes to inject records into the system, run the following:

docker-compose -f docker-compose.probes.yaml run gratia-probe-pbs
docker-compose -f docker-compose.probes.yaml run gratia-probe-condor

or

docker-compose -f docker-compose.probes.yaml up -d

to run them all.
